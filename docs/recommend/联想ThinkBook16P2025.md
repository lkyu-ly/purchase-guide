## 优缺点[<Icon icon="clarity:info-line" />](/recommend/推荐#优缺点)

|                  优点                  |           缺点           |
| :------------------------------------: | :----------------------: |
| 售后非常好，上门服务与意外险和数据拯救 |      没有 USB4 接口      |
