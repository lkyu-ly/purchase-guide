## 适合人群

需要一台性价比**非常高**，性能释放不错，拓展较强的游戏本，对售后与重量不那么敏感，对高帧率网游刚需较大，且偏爱 Intel 处理器。

## 总结

极光 X 延续了 24 年的性价比路线，使用 N-1 与 N-2 CPU 拉低整套配置的价格，但性能上并没有落于下风，在电竞网游中依旧表现不错。这套模具的内吹散热结构表现不错，功耗策略并不激进，噪音调教不让人烦躁。搭配这块各方面参数都称得上优秀的屏幕，这台机器的整体使用体验并没有因为其性价比而有所缺失。

不过 HX 处理器的外围功耗相对较高，机器的离电续航表现不会太亮眼。同时硬盘使用的 QLC 颗粒在寿命和长时读写性能上也注定是不如 TLC 颗粒。

同时，对于这一系列同显卡但 CPU 不同的机型，14650HX 和 13700HX 在同显卡的情况下表现差异微乎其微，仅在 PUBG 等内存敏感的应用场景会有些许差距，具体差异可以参考我们前面列出的处理器完整规格表。
