## 优缺点[<Icon icon="clarity:info-line" />](/recommend/推荐#优缺点)

|              优点              |                缺点                 |
| :----------------------------: | :---------------------------------: |
| 机身与适配器相对轻便，方便携带 |    性能释放稍弱，游戏性能相对弱     |
|  接口与拓展性丰富，屏幕素质好  | 内存为单根 32G，性能较两根 16G 要低 |
|   续航相对不错，机器噪音较低   |              网卡略差               |

## 适合人群

需要一台性价比较高，拓展性较强的轻薄低噪音游戏本，对售后没有太大的需求，没有生产力需求，并且主玩 3A 单机游戏。
